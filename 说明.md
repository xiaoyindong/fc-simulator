#### 安装依赖

```s
npm install
```

#### MAC

将下载好的nwjs.app文件copy到fc-simulator目录下，双击运行nwjs.app，将games/Super Mario Bros. (World).nes拖拽进去即可。

按键: wasd,jk

支持nes的fc游戏，可在百度自行下载。仅支持部分，未完全支持。

#### windows

将fc-simulator目录下的文件与windows下载的nwjs放在同一文件夹，双金nwjs.exe即可。


#### 其他

如不想基于nw可下载浏览器运行版本。https://github.com/xiaoyindong/fc-simulator2
